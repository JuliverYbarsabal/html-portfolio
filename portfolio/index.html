Life-log Geo Map System

This system will help support someone, especially with Dementia with their daily activity since the system uses Artificial Intelligence(AI) to learn certain habits and cues which will overall, support user with their memory and reasoning by knowing which photos and more information they have recently taken to trigger memories.

Functionality includes:

Basic features:

  • Added several unique events into my file. My program currently holds 23 events with each one having their own information
  • Created a Windows form GUI using Visual Studio and made a canvas area/map to display the events from your SOAP file.
  • Used LINQ to XML to load the data from the SOAP file into an appropriate collection of objects. While also using a C# Dictionary as the format for my collection. In addition, I also develop an appropriate class for the events.
  • Displayed the events (using suitable icons, text etc) on the canvas
  • Gave the user the option to Add a new event or Retrieve information by clicking on the canvas. Overall, If adding, my system should use a sub form to obtain the relevant information and insert into the SOAP file. If retrieving, I should develop a method to indicate which of the current events are being accessed and display the information.

Weather estimation program Object-Oriented project

C++ · OOP · Data Structures

Weather estimation program using historical data obtained in perth city 1998 - 2020. object-oriented solution and implementation using c++

Functionality includes:

Basic features:

  • The average wind speed and sample standard deviation for the wind speed given a specified month and year.
  • Average ambient air temperature and sample standard deviation for each month of a specified year
  • Total solar radiation in kWh/m^2 for each month of a specified year.
  • Average wind speed(km/h), ambient air temperature and total solar radiation in kWh/m^2 for each month of a specified year. The standard deviation is printed in () next to the average in a file named 'WindTempSolar.csv
  • Times for the highest solar radiation in kWh/m2 of a specified date.
  • Exit the program.
